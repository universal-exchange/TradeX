### 项目概述
交易服务特性：
+ 基于 CyberX 高性能分布式异构计算框架 灵活构建；
+ 提供适配 C++、Python、JavaScript 等开发语言的接口和示例；
+ 广泛支持国内国外证券经纪商各类股票、期货、期权、指数、基金等的交易接口；
+ 灵活支持直接委托下单、交易中心处理、交易服务代理等不同时效性和扩展性的架构模式。
